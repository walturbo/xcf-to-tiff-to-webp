# xcf-to-tiff-to-webp
Partendo da un file .XCF creato con Gimp, questo script ti permette di convertire prima in TIFF (8-bit o 16-bit per canale colore), senza perdere qualità. Dopo ti converte il file TIFF in formato WebP. Potrai scegliere la qualità e 5 diversi formati di output personalizzabili.
